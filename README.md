# NoteKeeper
Notes app built with the MERN stack & Redux.

## Usage
Rename the .envexample file to .env and add your MONGODB_URI.

## Installation

#### Install backend dependencies
```bash
npm install
```

#### Install frontend dependencies
```bash
cd frontend
npm install
```

## Run 

#### Backend (:3001)
```bash
npm run server
```

#### Frontend (:3000)
```bash
npm run client
```
